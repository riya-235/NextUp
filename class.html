Coding Courses

Available Courses

Intro to Python

A 6-week online Python programming workshop. This beginner-friendly course introduces the fundamentals of Python programming over six sessions. Students will learn essential concepts such as variables, data types, loops, conditionals, and functions, ensuring a strong foundation in Python. By the end of this course, students will be able to write basic scripts and develop simple programs, preparing them for further exploration in programming and data science.

Course Dates

Every Wednesday starting from 10/23 - 12/4 (Excluding Thanksgiving Week 11/27)

Intro to Website Development

A 6-week online Website Development programming workshop. This course introduces students to the basics of web design using HTML, CSS, and JavaScript. Students will learn how to create structured web pages with HTML, style them using CSS, and add interactive elements with JavaScript. By the end of this course, participants will have a solid foundation in web design and be able to create visually appealing, functional web pages, as well as their own finished website.

Course Dates

Every Tuesday starting from 10/22 - 12/3 (Excluding Thanksgiving Week 11/26)
